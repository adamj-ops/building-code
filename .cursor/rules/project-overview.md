# Minnesota Building Code Database - Project Overview

## Mission Statement

Build a comprehensive, searchable database of Minnesota building codes, local amendments, permit requirements, and fee schedules across all 87 counties and 853+ cities. Provide AI-powered search and interpretation to help contractors, developers, and homeowners navigate complex regulatory requirements.

## Project Status

**Current Phase**: MVP Development (Phase 1-2)
**Started**: January 2026
**Target MVP**: Q1 2026

## Core Value Propositions

1. **Centralized Information** - Single source of truth for MN building codes
2. **Jurisdiction-Specific Data** - Local amendments, fees, contacts for each city
3. **AI-Powered Search** - Natural language queries with intelligent summaries
4. **Time Savings** - Reduce research time from hours to seconds
5. **Accuracy** - Verified, up-to-date information with source citations

## Key Stakeholders

- **Build Co.** - Primary internal user for project planning
- **Contractors** - Need permit requirements and fee estimates
- **Developers** - Need code compliance information across jurisdictions
- **Homeowners** - Need to understand DIY permit requirements

## Success Metrics

- Search query response time < 2 seconds
- 95%+ data coverage for metro area jurisdictions
- User satisfaction score > 4.5/5
- Monthly active users (target: 500+ for internal use)
